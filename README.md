## Publicar en GitHub Pages

1. Copiar `index.html`, `estilos.css`, `README.md` y la carpeta `descargas` en la raíz del repositorio.
2. Subir los programas dentro de `descargas` con los nombres exactos.
3. Hacer commit y push.
4. Activar GitHub Pages desde `Settings > Pages > Deploy from branch > main / root`.

## Instalación

La página incluye instrucciones para Windows y Android.

- En Windows puede aparecer una advertencia de seguridad. Se indica usar `Más información` y luego `Ejecutar de todos modos` cuando el archivo se descargó desde una fuente confiable.
- En Android, si el APK se recibe por WhatsApp o se descarga fuera de Play Store, puede pedir permiso para instalar aplicaciones desconocidas. Se indica activar `Permitir desde esta fuente` para WhatsApp, Chrome o el administrador de archivos usado.
