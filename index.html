Cómo instalar

Indicaciones para Windows y Android

Estas indicaciones ayudan cuando el programa se descarga desde la web, GitHub o se recibe por WhatsApp.

Windows

Instalar en PC

  1. Descargá el ZIP del programa para PC.
  2. Extraé el ZIP en una carpeta.
  3. Ejecutá el instalador o el archivo indicado dentro del paquete.
  4. Si Windows muestra una advertencia, elegí Más información y luego Ejecutar de todos modos.
  5. Si aparece el Control de cuentas de usuario, presioná Sí para permitir la instalación.

La advertencia puede aparecer porque es un programa propio y no viene desde Microsoft Store. Instalalo solo si lo descargaste desde esta página o desde un contacto confiable.

Android

Instalar APK recibido por WhatsApp o descargado

  1. Descargá el archivo Android. Si viene dentro de un ZIP, primero extraelo desde el administrador de archivos del teléfono.
  2. Tocá el archivo .apk para iniciar la instalación.
  3. Android puede avisar que la app no viene de Play Store. Entrá en Ajustes cuando lo pida.
  4. Activá Permitir desde esta fuente para WhatsApp, Chrome o el administrador de archivos que estés usando.
  5. Volvé a tocar el APK y presioná Instalar.

Esta autorización se pide porque la aplicación se instala manualmente y no desde Play Store. Después de instalar, podés volver a desactivar ese permiso si querés.

Cómo publicar

Pasos rápidos

  1. Copiá index.html, estilos.css, README.md y la carpeta descargas en la raíz del repositorio.
  2. Dentro de descargas, subí los ZIP con los nombres exactos de la tabla.
  3. Subí los cambios a GitHub.
  4. Activá GitHub Pages desde Settings → Pages → Deploy from branch → main / root.
  5. Probá la página y cada botón de descarga.
