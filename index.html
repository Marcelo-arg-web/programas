Instalación

Guía rápida para instalar

Pasos básicos para instalar los programas descargados.

Windows

Instalar en PC

  1. Descargá el programa para PC.
  2. Extraé el archivo ZIP.
  3. Ejecutá el instalador o el archivo indicado.
  4. Si Windows muestra una advertencia, elegí Más información y después Ejecutar de todos modos.
Android

Instalar APK

  1. Descargá el archivo Android.
  2. Si viene dentro de un ZIP, extraé el APK.
  3. Tocá el archivo APK para instalar.
  4. Si Android pide autorización, activá Permitir desde esta fuente para WhatsApp, Chrome o Archivos.
